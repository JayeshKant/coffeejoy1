Demo Project for the SE practical work.
Start by opening main_project.pro with Qt Creator.

You're free to re-use code for your own work.
Please make sure that there are no unnecessary files left of the demo when you hand in the final version of your project.

The demo showcases some aspects that you will encounter during work on your own project:
 - basic project structure and build system
 - bootstrapping an application
 - configuration files and command line arguments
 - dependency injection
 - lose coupling through interfaces
 - use of Signals and Slots
 - use of simulated time and timers
 - a basic GUI
 - Qt resource access
 - testing setup
 - mock-ups
 - ...
